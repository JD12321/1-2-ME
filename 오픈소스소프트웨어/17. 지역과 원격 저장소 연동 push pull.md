## 2) 지역에서 원격 저장소로 push
* * *
- - -
### (1) 지역 저장소에서 push
#### ◈ 쓰기 권한이 있어야 push가 가능
   - 자신의 저장소나 다른 사람의 저장소라면 협업자(collaborator)로 등록

#### ◈ Push
   - 로컬 저장소에서 남겨놓은 코드 변경 이력들이 원격 저장소로 전송

#### ◈ 명령
   - $ git push <저장소별칭명> <브랜치명>
      - $ git push origin topic
