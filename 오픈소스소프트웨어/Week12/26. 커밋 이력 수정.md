### (1) 최근 커밋 메시지 수정
#### ◈ 기본 편집기 설정 방법
  - $ git config --global core.editor ‘code --wait’
#### ◈ 설정된 편집기로 최근 커밋 메시지 수정
  - $ git commit --amend

    - 새로운 커밋 ID로 수정됨
#### ◈ 최근 커밋 메시지를 직접 입력해 수정
  - $ git commit --amend -m "an updated commit message“

    - 새로운 커밋 ID로 수정됨
